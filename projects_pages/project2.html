🛒 E-Commerce Web Application (Next.js, MongoDB, Cloudinary) 

A full-stack online store built with Next.js, featuring secure user authentication, dynamic product management, and cloud-based media storage.
This project demonstrates modern web development practices with a focus on scalability, performance, and clean UI/UX.

Key Features 

🧩Full-Stack Functionality: Integrated MongoDB for data persistence and Cloudinary for image management.
⚡Next.js Framework: Server-side rendering and API routes for efficient data handling.
🛍️Product Management: Admin dashboard for managing product listings and user data.
🌈Responsive Design: Fully responsive and optimized for desktop and mobile.
☁️Cloud Integration: Product images are securely stored and served from Cloudinary.
🔐Authentication & Authorization: Users must sign in to view products; admins have full CRUD access (add, update, delete).

 Tech Stack

Frontend: Next.js 
Backend: Next.js API routes, Node.js
Database: MongoDB / Mongoose
Media Storage: Cloudinary
Authentication: JWT / NextAuth
