🌐 DevConnect – Full-Stack Portfolio Sharing Platform 

🌐 DevConnect – Full-Stack Portfolio Sharing Platform (HTML, CSS, JavaScript, PHP, MySQL)

A professional networking platform inspired by LinkedIn, built completely from scratch using HTML, CSS, JavaScript, and PHP.  
Users can create profiles, upload their projects, explore others’ work, and connect with developers around the world.

Key Features

👤 User Profiles: Each user has a personalized profile page displaying bio, skills, and uploaded projects.  
📂 Project Uploads: Users can upload and manage their projects, including titles, descriptions, and media.  
💬 Interactive Feed: A central feed where all recent uploads and updates from users appear dynamically.  
🔍 Search & Discover: Built-in search system to find users or projects by name, technology, or tags.  
🗄️ PHP + MySQL Backend: Handles user data, authentication, and file uploads securely.  
📱 Responsive Design: Fully optimized layout that works seamlessly on desktop and mobile.  
🧠 Dynamic Frontend: JavaScript enhances interactivity — live form validation, animations, and real-time updates.

Tech Stack

Frontend: HTML, CSS, JavaScript  
Backend: PHP  
Database: MySQL  
Functionality: User profiles, project uploads, feed system, authentication, search  
Design: Modern, clean, and responsive layout
