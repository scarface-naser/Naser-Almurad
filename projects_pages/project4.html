💰 SmartBudget – Personal Finance Tracker (React, Firebase, Material-UI)

A modern and interactive budget management app built with React, Firebase, and Material-UI.  
SmartBudget helps users easily add, delete, and analyze their expenses while automatically calculating total spending in real time.

Key Features

📤 Add & Manage Transactions: Users can send, retrieve, and delete expense data directly from Firebase.  
⚙️ Auto Total Calculation: Automatically updates total balance whenever an item is added or removed.  
🌙 Dark & Light Mode: Includes a toggle between light and dark themes for better user experience.  
📊 Responsive Dashboard: Built with Material-UI components such as AppBar, Card, Grid, and Stack for a clean layout.  
🖼️ Sidebar & Modal Controls: Sidebar can be shown or hidden; modals and tooltips enhance interactivity.  
📋 Form Validation: Integrated React Hook Form for professional form handling and validation.  
💡 Real Project Experience: Covered complete workflow — from setup and UI design to debugging and polishing.

Tech Stack

Frontend: React.js  
UI Library: Material-UI (MUI)  
Backend: Firebase (Realtime Database)  
Forms: React Hook Form  
Features: Add/Delete transactions, Auto Total, Dark Mode, Sidebar & Modal UI  
Design: Responsive, modern, and user-friendly
