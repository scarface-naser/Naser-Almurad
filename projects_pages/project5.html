🛍️ Redux Store – React & Express E-Commerce 

🛍️ Redux Store – Full-Stack E-Commerce App (React, Redux, Express, MongoDB)

A modern e-commerce platform built with React, Redux, Express, and MongoDB.  
Users can browse products, add them to a shopping cart, and manage their orders with real-time state management.

Key Features

🛒 Redux State Management: Efficiently manages global state for products, cart, and user sessions.  
⚡ Express & MongoDB Backend: REST API handles product data, orders, and user authentication.  
📦 Product Management: Add, update, and delete products from the store (admin panel included).  
💳 Shopping Cart: Add items, update quantities, and calculate totals dynamically.  
🔐 Authentication & Authorization: Users must sign in to purchase; admins have full control over products and orders.  
📱 Responsive UI: Clean, modern design optimized for desktop and mobile.  
🌐 API Integration: Frontend communicates with Express API for CRUD operations in MongoDB.

Tech Stack

Frontend: React.js, Redux  
Backend: Express.js, Node.js  
Database: MongoDB / Mongoose  
State Management: Redux  
Features: Shopping cart, user authentication, admin CRUD, real-time updates  
Design: Responsive, modern, and user-friendly
